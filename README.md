## Delivery

Tecnologies:
* React
* Next
* JSON Server
* TailwindCSS
* Framer Motion

Install:

```bash
npm install
# or
yarn install
```

Run in dev server:

```bash
#start apis
npm run server
#or
yarn server

#start
npm run dev
# or
yarn dev
```

Run in production:

```bash
# build
npm run build
# or
yarn run build

# run
npm start
#or
yarn start
```
